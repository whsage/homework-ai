#### **小学数学辅导（6-12岁）**

**教学特点：**
- 🍕 生活场景教学
- 📊 图形化展示
- 🎯 鼓励式引导
- 🎮 趣味互动

**辅导内容：**
- 加减乘除运算
- 分数、小数、百分数
- 简单几何图形
- 应用题解答
- 时间和长度单位
- 简单统计图表

**真实案例：**
```
问题：什么是 1/2？

传统方法：
"1/2就是一个整体分成两份，取其中一份"

我们的方法：
🍕 "想象一个披萨，你和朋友一起吃，每人吃一半，你吃的就是1/2"
🎂 "生日蛋糕切成两块，你吃一块，就是吃了1/2个蛋糕"
🍎 "一个苹果分给两个人，每人得到1/2个苹果"

效果：通过生活场景，孩子自然理解了分数概念
```

**理论支撑：**
基于皮亚杰认知发展理论，7-11岁儿童处于具体运算期，需要具体事物支持思维。

#### **初中数学辅导（12-15岁）**

**教学特点：**
- 💡 概念深度理解
- 🔗 知识体系构建
- 🎯 逻辑推理训练
- 🔍 多角度分析

**辅导内容：**
- 代数：方程、不等式、函数
- 几何：三角形、圆、相似
- 统计：概率、数据分析
- 数与式：因式分解、根式
- 图形变换：平移、旋转、对称
- 应用题：行程、工程、经济

**真实案例：**
```
问题：为什么二次函数图像是抛物线？

苏格拉底式引导：
AI: "你知道 y = x² 这个函数吗？"
学生: "知道"
AI: "当 x = 1 时，y 等于多少？"
学生: "y = 1"
AI: "当 x = 2 时呢？"
学生: "y = 4"
AI: "当 x = -1 时呢？"
学生: "y = 1"
AI: "你发现什么规律了吗？"
学生: "正数和负数的 y 值相同！"
AI: "对！这就是为什么图像关于 y 轴对称..."

效果：通过引导，学生自己发现了抛物线的对称性和形状特征
```

**理论支撑：**
基于维果茨基最近发展区理论和布鲁纳发现学习理论，通过引导让学生在适当难度下自主发现知识。

#### **高中数学辅导（15-18岁）**

**教学特点：**
- 🔬 抽象思维培养
- 📐 多种解法对比
- 🎯 数学思想渗透
- ⚡ 解题规律总结

**辅导内容：**
- 函数与导数：极值、单调性、切线
- 三角函数：恒等变换、图像性质
- 立体几何：空间向量、体积面积
- 解析几何：直线、圆、圆锥曲线
- 数列：等差、等比、递推
- 概率统计：排列组合、分布、期望

**真实案例：**
```
问题：导数 f'(x) 的几何意义是什么？

深度引导：
AI: "你知道函数图像上某一点的切线吗？"
学生: "知道"
AI: "切线的斜率代表什么？"
学生: "代表函数在该点的变化率"
AI: "很好！现在思考：当 Δx 越来越小时，割线会变成什么？"
学生: "会变成切线！"
AI: "没错！所以导数就是切线的斜率。现在你能理解为什么 f'(x)>0 时函数递增吗？"
学生: "因为切线斜率为正，函数在上升！"

效果：通过层层递进，学生深刻理解了导数的几何意义和应用
```

**理论支撑：**
基于布鲁姆教育目标分类学的分析和评价层次，培养高级认知能力。

#### **大学数学辅导（18岁以上）**

**教学特点：**
- 📚 严格的数学证明
- 🧮 理论推导训练
- 🌐 实际应用场景
- 🔍 研究性学习

**辅导内容：**
- 高等数学：极限、微积分、级数
- 线性代数：矩阵、向量空间、特征值
- 概率论：随机变量、分布、大数定律
- 数理统计：参数估计、假设检验
- 常微分方程：求解方法、应用
- 复变函数：解析函数、留数定理

**真实案例：**
```
问题：如何理解 lim(x→a) f(x) = L 的严格定义？

理论引导：
AI: "你知道极限的直观含义吗？"
学生: "x 趋近于 a 时，f(x) 趋近于 L"
AI: "对，但'趋近'不够严格。我们需要量化。如果我说 f(x) 与 L 的距离小于 0.01，你能找到一个 x 的范围吗？"
学生: "可以，让 |x-a| 足够小"
AI: "很好！现在把 0.01 换成任意小的 ε，你需要找到对应的 δ，使得：当 0<|x-a|<δ 时，|f(x)-L|<ε"
学生: "哦！这就是 ε-δ 定义！"

效果：通过从具体到抽象，学生理解了严格定义的必要性和含义
```

**理论支撑：**
基于布鲁姆分类学的最高层次（创造和评价），培养独立研究能力。
