## 💡 建议新增的通知类型

### 1. 学习习惯类 📅

#### 1.1 连续学习打卡 🔥
**触发时机：** 连续多天完成作业
```
🔥 连续学习 3 天！
太棒了！你已经坚持学习 3 天了。继续保持，7 天连续学习徽章在等你！
```

**里程碑：** 3天、7天、14天、30天、60天、100天

#### 1.2 学习时段偏好 🌙
**触发时机：** 检测到用户的学习时段规律
```
🌙 夜猫子学习者
我们发现你经常在晚上 8-10 点学习，这是你的高效时段！保持这个好习惯。
```

**类型：**
- 早起学习者（6-8点）
- 午后学习者（14-16点）
- 夜猫子学习者（20-22点）

#### 1.3 学习频率提醒 ⏰
**触发时机：** 用户 3 天没有学习
```
👋 好久不见！
已经 3 天没看到你了，今天要不要来做个作业？你的学习伙伴在等你！
```

**提醒级别：**
- 3 天：温馨提醒
- 7 天：鼓励回归
- 14 天：特别关怀

## 🎨 通知设计原则

### 1. 及时性
- 成就达成后立即推送
- 不要延迟超过 1 分钟

### 2. 个性化
- 使用用户昵称
- 根据学习数据定制内容

### 3. 正向激励
- 多用鼓励性语言
- 避免负面表达

### 4. 适度频率
- 每天最多 3-5 条通知
- 避免打扰用户

### 5. 可操作性
- 提供明确的行动指引
- 点击通知可跳转到相关页面
